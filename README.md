### 初始化依赖配置

```
yarn install
```

### 开发环境 启动运行

```
yarn run dev
```

### 测试环境 打包构建

```
yarn run  build:test
```

### 生产环境 打包构建

```
yarn run build:prod

yarn analyz // 打包分析
```

### 自动化测试

```
yarn run test
```

### Lint 检查并修复

```
yarn run lint
```
